Understanding Dictionary Comprehensions in Python 3

Date Tue 23 July 2019

Introduction

Dictionary comprehensions is a feature in Python that allows us to create dictionaries based on an iterable in a concise and easy to read manner.

The use of dictionary comprehensions is done to limit the lines of code in Python programs, in addition to enhancing readability for a better development experience.

Dictionary Comprehensions

As a reminder, to create a simple dictionary in Python, we use the following syntax:

numbers = {'one': 1, 'two': 2}
print(numbers)

Here, the strings 'one' and 'two' are called keys, while the integers 1 and 2 are their respective values. For more about dictionaries in Python, see Understanding Dictionaries in Python 3.

In the code above, we directly supply the keys and their values to the dictionary. With dictionary comprehensions, we can use keys and values from another iterable as follows:

numbers = [('one', 1), ('two', 2)]

numbers_dict = {key: value for key, value in numbers}

print(numbers_dict)

As you can see in the second line above, we use a for clause to loop through the numbers list and extract keys and their values from the inner tuples in the list, the first element in each tuple is the key, which is stored each time in the key variable surrounding the for clause, while the second element is its value, which gets stored in the value variable each time we iterate through the iterable list number.

In the last line, we use the print() function to print the dictionary numbers_dict we created using the dictionary comprehension, the output is as follows:

{'one': 1, 'two': 2}

If we use a simple for loop instead of a dictionary comprehension, we would have to write the code as follows:

numbers = [('one', 1), ('two', 2)]

numbers_dict = dict()
for key, value in numbers:
    numbers_dict[key] = value

print(numbers_dict)

We had to start by creating the empty dictionary numbers_dict in the second line using the built-in dict() function, then we iterate through the iterable list numbers. The for loop then goes through the tuples in the list one by one, storing the first item in the key variable, and the second item in the value variable. Each key and value is then added to the numbers_dict dictionary using the assignment operation numbers_dict[key] = value.

The output of the above code would be the same as the dictionary comprehension we wrote earlier:

{'one': 1, 'two': 2}

Modifying Keys and Values in a Dictionary Comprehension

We can have any valid Python expression in the dictionary comprehension, this allows us to modify keys and their values so they appear in a certain way on the outputted dictionary. For example, we can use the capitalize() method to capitalize the keys:

numbers = [('one', 1), ('two', 2)]

numbers_dict = {key<^>.capitalize()<^>: value for key, value in numbers}

print(numbers_dict)

{'One': 1, 'Two': 2}

We can also use the same concept to transform values, for example, we can convert values from integers to strings by using the str() built-in function:

numbers = [('one', 1), ('two', 2)]

numbers_dict = {key.capitalize(): <^>str(value)<^> for key, value in numbers}

print(numbers_dict)

{'One': '1', 'Two': '2'}

Using Functions in Dictionary Comprehensions

Similarly, we can define and use a function that receives some input and returns a modified version of it to manipulate keys and values in the resulting dictionary:

def duplicate(value, times):
    return value*times


shapes = [('stars', '*'), ('hashes', '#')]

shapes_dictionary = {key.upper(): duplicate(value, 5)
                     for key, value in shapes}

print(shapes_dictionary)

Here, we create a function duplicate() that receives two parameters, value and times, and returns the result of the simple Python operation value*times, for example calling the function duplicate('A', 5) would result in the string 'A' being duplicated 5 times, which gives us the string 'AAAAA' as a return value. Although this function can also serve as a multiplier when using integers, therefore duplicate(5, 5) returns the result of the multiplication operation 5*5, which equals the integer 25. However, we will only use this function to duplicate strings in this tutorial.

Note that we can break the dictionary comprehension into multiple lines to enhance readability with appropriate indentation.

The created dictionary, shapes_dictionary, will have uppercase keys due to the usage of the upper() method, and the values will be duplicated five times because we called the duplicate() function with the value as a first argument, and the integer 5 as the duplication variable. The output is as follows:

{'STARS': '*****', 'HASHES': '#####'}

As you can see, we were able to manipulate both keys and values inside the dictionary comprehension.

Using Conditionals in Dictionary Comprehensions

We can control what keys and what values can appear in the created dictionary using an if statement in the dictionary comprehension. For example, we can restrict the dictionary keys to be only 3 characters long or shorter, and reject any key if it has more than 3 characters:

names = [('John', 'Doe'),
         ('Bob', 'Brown'),
         ('Ali', 'David'),
         ('Alexander', 'Young'),
         ('Jennefer', 'Armstrong'),
         ]

names_dict = {firstname: lastname
              for firstname, lastname
              in names
              if len(firstname) <= 3
              }

print(names_dict)

In the above code, we use the condition len(firstname) <= 3 to indicate that the resulting dictionary can only have entries where the first name length is equal to or shorter than three characters long. The output is as follows:

{'Bob': 'Brown', 'Ali': 'David'}

Using The zip() Function to Create a Two-value Iterable

Although we can use any iterable in a dictionary comprehension in order to create a dictionary, it is common practice to rely on an itrable that holds pairs of values, this way we can use one item in the pair as the key, and the other to serve as its value.

Python provides us with the built-in zip() function that accepts two iterables and combines them into an iterable that holds pairs of values, each item in the pair comes from each of the two iterables that were passed.

For example, Let's say we have a list firstnames containing first names, and a list lastnames that contains the last names for each item in the other list like so:

firstnames = ['John', 'Sammy']
lastnames = ['Doe', 'The Shark']

We can use the zip() function to zip the two lists, the result will be an iterator that contains several Python tuples, each tuple will have the first name and the last name like so:

firstnames = ['John', 'Sammy']
lastnames = ['Doe', 'The Shark']

names = zip(firstnames, lastnames)

for name in names:
    print(name)

The output will draw a clear picture of how this can be useful in practice:

('John', 'Doe')
('Sammy', 'The Shark')

As you can see, the zip() function is handy when writing a dictionary comprehension that relies on two iterables.

Here is an example where we use zip() in a dictionary comprehension converting the last name into uppercase letters with the upper() method:

firstnames = ['John', 'Sammy']
lastnames = ['Doe', 'The Shark']

names = {firstname: lastname.upper()
         for firstname, lastname
         in zip(firstnames, lastnames)
         }
print(names)

The output is as follows:

{'John': 'DOE', 'Sammy': 'THE SHARK'}

Conclusion

In this tutorial, we have learned how to use dictionary comprehensions to write more concise and readable Python code.

However, dictionary comprehensions can sometimes make our code less readable, especially when using long expressions, it is important to maintain readability if a dictionary comprehension is too long by breaking it into multiple parts.
